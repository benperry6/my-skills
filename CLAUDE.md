# Skills Repo — Règles locales

## Commit & push obligatoires — AUCUNE EXCEPTION
Tout changement dans ce repo (création, modification, suppression de fichier, merge, résolution de conflit) doit être **immédiatement** commité ET poussé sur origin dans la même action.
Ne JAMAIS laisser de changements locaux non poussés. Ne JAMAIS attendre une validation utilisateur pour commit+push dans ce repo — c'est autorisé par défaut ici.
Si un push échoue, le signaler immédiatement.

`AGENTS.md` est un symlink vers `CLAUDE.md`. `CLAUDE.md` est le fichier réel (source de vérité).
