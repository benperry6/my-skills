# Templates Prêts à Coller

## Emails

### Email Court - Express7 (Promo/Annonce)

```
Objet: {bénéfice spécifique en X temps}

Tu veux {bénéfice} sans {pain principal}?

C'est ce que {script/méthode/outil} fait pour {segment cible}.

Aujourd'hui tu peux l'utiliser ici: {lien/CTA}

Dispo jusqu'à {deadline}.

→ {CTA}
```

### Email Curiosité + WIIFM

```
Objet: {bénéfice spécifique} sans {pain majeur}

Comment {personne/segment} encaisse {résultat} en {délai}...

...alors que tout le monde galère avec {problème commun}.

Vu dans {source} / déjà appliqué sur {segment}.

Accède ici: {lien}

Offre valable jusqu'à {date/stock}.
```

### Email Relance Futsal (Trafic Chaud)

```
Objet: Tu passes déjà les {seuil €} par mois?

{Hook/Question + promesse courte}

Ex: Antonio est passé de Leclerc à 70k€/mois avec {méthode}.

Je lance un groupe pilote de {X} places.

→ {lien}

Clôture ce soir / {date}.

PS: {garantie/accès}
```

## Pages de Vente

```
- {Élément 1}
- {Élément 2}
```

## Upsells (OTO)

### Upsell Standard

```
✅ Bonne nouvelle: ta commande est validée.

Avant d'accéder à {produit initial}, ajoute {Upsell} pour {bénéfice spécifique}.

**Ce que tu ajoutes:**
- {Point 1}
- {Point 2}
- {Point 3}

Prix: {€}

[Oui, j'ajoute maintenant]   [Non, je passe]
```

### Upsell avec Urgence

```
✅ Commande validée.

**Offre unique - Cette page seulement**

Ajoute {upsell} pour {bénéfice} en un clic.

Tu ne reverras plus cette offre à ce prix.

Prix: {€} (au lieu de {prix normal})

[Oui, j'ajoute]   [Non, merci]
```

```
**Ce que tu reçois aujourd'hui:**

✓ {Module A} ........................ valeur {€}
✓ {Module B} ........................ valeur {€}
✓ Bonus {1} ......................... valeur {€}
✓ Bonus {2} ......................... valeur {€}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Valeur totale: {ancrage €}

**Aujourd'hui: {prix réel €}**
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### ABL (Always Be Leaving)

```
Honnêtement, ma vie ne change pas si tu passes ton tour.

Avec ou sans toi, le programme tourne.

Si tu veux profiter de {mécanisme/Big Idea} maintenant, clique.

Sinon, rien ne m'oblige à te retenir.

→ [CTA]
```

### Close "Deux Catégories"

```
Certaines personnes restent bloquées dans {douleur}.

D'autres décident de {bénéfice du produit}.

Tu es de quel côté?

→ [CTA]
```

### Hook Big Idea

```
Je n'aurais jamais imaginé que {source inattendue: CIA, Bible, Casino}
deviendrait la clé pour {bénéfice précis}.

Découverte en {année/contexte}, testée sur {X} cas,
et applicable dès ce soir.

Curieux?
```

### Bloc Mécanisme Unique

```
Le {Nom mécanisme} = {renommage court}.

Né de {origine}, il remplace {ancienne croyance}
par {nouvelle règle}.

Résultat: {bénéfice mesurable} même si {objection courante}.
```

## Réponses Express aux Objections

| Objection | Réponse template |
|-----------|------------------|
| **Légal/éthique?** | "Oui, si tu suis ces étapes. Rocket Internet en vit depuis 2007." |
| **Temps?** | "Objectif 28 jours: méthode pré-testée, checklists fournies." |
| **Saturation?** | "On copie la structure, pas les mots; chaque niche reste différente." |
| **Coût de lancement?** | "Lancement frugal: {éléments} = moins de {X}€." |
| **Ça marche vraiment?** | "{Preuve}: {cas/chiffre} en {délai}." |
