# My Custom Skills

Personal skills for AI coding assistants (Claude Code, Codex, Gemini CLI, etc.)

## Skills

| Skill | Description |
|-------|-------------|
| `experiential-offer-design` | Design products/services as memorable experiences with unboxing, onboarding, rituals, and segmented messaging |

## Installation

```bash
npx skills add benperry6/my-skills
```

## Structure

Each skill follows the standard structure:
```
skill-name/
├── SKILL.md           # Main skill file (required)
└── references/        # Supporting documentation (optional)
```
